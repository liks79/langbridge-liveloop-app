# Contributing to LangBridge

Thank you for your interest in contributing to LangBridge! We welcome all contributions, from bug reports and feature requests to code changes and documentation improvements.

## How to Contribute

1. **Fork the Repository**: Create your own copy of the repository.
2. **Clone the Fork**: `git clone https://github.com/YOUR_USERNAME/langbridge-liveloop-app.git`
3. **Create a Feature Branch**: `git checkout -b feat/your-feature-name`
4. **Make Your Changes**: Implement your feature or fix.
5. **Run Tests**: Ensure all tests pass with `npm test`.
6. **Commit Your Changes**: `git commit -m "feat: your descriptive message"`
7. **Push to Your Fork**: `git push origin feat/your-feature-name`
8. **Open a Pull Request**: Submit your PR to the `main` branch of the original repository.

## Code Style

- We use **React 19** and **Tailwind CSS 4**.
- Please follow the existing coding style and use descriptive variable names.
- Ensure that you don't commit any sensitive information (API keys, etc.).

## Reporting Issues

If you find a bug or have a suggestion, please open an issue on GitHub with as much detail as possible, including steps to reproduce the issue.
